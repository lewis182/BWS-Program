# BWS 10-Week Chest Program — Free Weights & Bench

A web-app workout tracker for the BWS 10-Week Chest Program (3-day split) by Jeremy Ethier,
with cable, machine, band and pull-up exercises swapped for free-weight and bench alternatives
listed in the original PDF (Pull-ups replaced by Barbell Bent-Over Row).

Open `index.html` — or visit the GitHub Pages URL — to use it. Progress is saved in your browser
via `localStorage` under key `bws-chest-program-free-weights-v1`.

Tutorial PDFs in `tutorials/` are pages extracted from the original programme by Jeremy Ethier.
